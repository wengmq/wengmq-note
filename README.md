# wengmq-note

## Getting Started

首先得有 node，并确保 node 版本是 10.13 或以上。

```bash
$ node -v
v10.13.0
```

进入项目目录，安装依赖

```bash
$ npm install
```

运行程序

```bash
$ npx dumi dev
```
